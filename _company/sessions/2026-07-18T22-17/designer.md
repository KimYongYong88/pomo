## 🚀 최종 실행 가이드라인 템플릿 구조 (Execution Guideline Template)

본 템플릿은 확정된 브랜드 시스템(딥 블루, Montserrat, Inter)을 기반으로, 사용자 입력 변수(채널, 시점)가 삽입되어 실제 배포에 즉시 사용될 수 있도록 설계되었습니다.

### 1. 디자인 시스템 정의 (Brand System Definition)

이 섹션은 모든 실행 가이드라인의 기준이 되는 브랜드 요소를 명확히 정의합니다.

| 요소 | 설정 값 | 용도 |
| :--- | :--- | :--- |
| **주요 색상** | Deep Blue (`#0047AB`) | 핵심 강조 및 배경 색상 |
| **보조 색상** | Light Gray / White | 본문 및 여백 |
| **헤드라인 폰트** | Montserrat (Bold/ExtraBold) | 제목, 핵심 메시지 강조 |
| **본문 폰트** | Inter | 상세 설명, 가이드라인 텍스트 |

### 2. 배포 실행 정보 입력 (Deployment Variables Input)

실제 실행 시 사용자로부터 입력받아 삽입될 변수 영역입니다. 이 부분은 사용자가 채우는 동적 필드임을 명시합니다.

| 변수 항목 | 삽입 위치 (Placeholder) | 설명 |
| :--- | :--- | :--- |
| **배포 채널** | `[CHANNEL_INPUT]` | 실제 콘텐츠가 배포될 플랫폼 (예: Instagram, YouTube, Email 등) |
| **배포 시점** | `[TIME_INPUT]` | 콘텐츠가 실제로 게시되거나 발송될 시간/날짜 (예: 2026년 7월 19일 오전 10시) |

### 3. 최종 실행 가이드라인 레이아웃 (Final Execution Guideline Layout)

실제 배포 실행을 위한 단계별 지침 및 시각적 요소가 통합되는 구조입니다. **모든 제목과 주요 섹션은 Montserrat를 사용하며, 상세 설명은 Inter를 사용하여 가독성을 극대화합니다.** 배경색은 Deep Blue 계열을 활용하여 권위와 신뢰감을 부여합니다.

#### **[템플릿 시작]**

**[섹션 1: 핵심 요약 및 목표 설정 (Header)]**
*   **제목:** `Montserrat ExtraBold` - [배포 채널] 콘텐츠 배포 실행 가이드라인
    *   *(Deep Blue 배경 또는 Deep Blue 강조선 적용)*
*   **핵심 목표:** `Inter Bold` - [TIME_INPUT]에 맞춰, [CHANNEL_INPUT] 플랫폼에서 시스템 아키텍트로서의 권위를 극대화하는 핵심 메시지 전달.

**[섹션 2: 디자인 및 비주얼 적용 (Visual Application)]**
*   **제목:** `Montserrat SemiBold` - 브랜드 시스템 반영 상세 지침
    *   *(Inter 본문 폰트로 상세 설명 시작)*
*   **컬러 팔레트 적용:** 배경은 Deep Blue, 강조 색상은 White/Light Gray를 사용하여 시각적 위계를 확보합니다. (Deep Blue: `#0047AB`)

**[섹션 3: 실행 단계별 지침 (Step-by-Step Execution)]**
*   **제목:** `Montserrat Medium` - 최종 배포 프로세스
    *   *(Inter 본문 폰트로 상세 설명 시작)*
*   **단계 1. 콘텐츠 검토 및 정렬:**
    *   `[CHANNEL_INPUT]`의 특성에 맞게 콘텐츠의 논리적 흐름을 재배치합니다. (최대 3단계 구조 권장)
*   **단계 2. 시각적 요소 조정:**
    *   모든 이미지, 인포그래픽에 Deep Blue를 주조색으로 적용하고, Montserrat 타이포그래피 계층(H1, H2, Body)을 엄격히 준수합니다.
*   **단계 3. 시간 기반 최적화:**
    *   `[TIME_INPUT]`의 특성(예: 아침/저녁 트래픽)을 고려하여 콘텐츠 노출 시간을 최적화하고, CTA(Call To Action)를 명확히 합니다.

**[섹션 4: 최종 점검 및 배포 (Final Review & Deployment)]**
*   **제목:** `Montserrat Bold` - 최종 승인 체크리스트
    *   *(Inter 본문 폰트로 상세 설명 시작)*
*   **체크리스트 항목:**
    1.  ✅ 브랜드 시스템(Deep Blue/Montserrat/Inter) 완벽 적용 여부: (Y/N)
    2.  ✅ 채널 특성 최적화 완료 여부: (Y/N)
    3.  ✅ 시점(`[TIME_INPUT]`) 기반 노출 시간 적합성: (Y/N)
    4.  ✅ ROI 및 데이터 신뢰도 확보 여부: (Y/N)

**[섹션 5: 실행 명령 (Execution Command)]**
*   **지침:** `Inter Bold` - 최종 배포 실행을 위한 명령어 준비
*   **실행 프레임워크:** 실제 배포 시, 본 가이드라인을 기준으로 [CHANNEL_INPUT]에 최적화된 포맷으로 결과물을 출력하십시오.
